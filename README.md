# MapWise v1

A simple travel map companion for saved places, addresses and navigation handoff.

## Features

- Interactive Leaflet / OpenStreetMap map
- Find current location
- Save current location
- Add pin by clicking the map
- Add place manually
- Saved places with categories
- Local-language address field
- Notes field
- Distance from current location
- Copy address / coordinates
- Large show-address mode
- Open destination in Organic Maps via geo: link
- Open destination in Apple Maps
- Google Maps fallback available in code
- Export / import saved places JSON
- Dark mode
- PWA support
- GitHub Pages ready

## Important

MapWise is not a full navigation app. It is a travel place manager. For turn-by-turn navigation, use Organic Maps / Apple Maps / another installed navigation app.

The map tiles load from OpenStreetMap while online. Saved places are stored locally and remain available after caching.
